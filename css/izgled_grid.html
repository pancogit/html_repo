Moguce je takodje koristiti i neke druge css funkcije koje se koriste za grid. Sadrzaj grid elemenata je moguce podesavati pomocu takozvane funkcije za fit-ovanje. Za slucaje fit-ovanja sadrzaja, koristicemo jedinice za sirinu koje su relativne u odnosu na broj 0 za zadati font (ch jedinica). Jedan ch iznosi sirinu jednog karaktera broja 0 za zadati font. S obzirom da se to razlikuje od fonta do fonta, tj. broj 0 nije isto sto i sirina karaktera slova (a, b, c...) za svaki font, koristicemo monospace font, zato sto je bas kod monospace-a sirina fonta za broj 0 ista kao i za slova, pa cemo dobiti da je na primer za 50ch sirine nekog grid elementa tacna duzina 50 karaktera, sto ne bi vazilo za druge font-ove.
